# Tower Defense

Простая игра Tower Defense на Python + Pygame.

## Как запустить

1. Установи зависимости:
pip install pygame

2. Запусти игру:
python main.py

## Как играть

- Кликай мышью чтобы ставить башни (стоит 50$)
- Башни автоматически стреляют по врагам в радиусе
- Не давай врагам дойти до конца — теряешь жизни
- Побеждай волны чтобы выжить

## Волны

| Волна | Враги | Особенность |
|-------|-------|-------------|
| 1 | 5 врагов | Обычные |
| 2 | 7 врагов | Больше HP |
| 3 | 8 врагов | Быстрые |
| 4 | 5 врагов | Толстые |

## Стек

- Python 3.10+
- Pygame 2.5
